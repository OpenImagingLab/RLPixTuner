# Goal Conditioned Reinforcement Learning for Photo Finishing Tuning


**NeurIPS 2024**

[Project Page](https://openimaginglab.github.io/RLPixTuner/) | [Video](https://www.youtube.com/watch?v=fFIkc3KHS28)


## Installation
```
pip install -r requirements.txt
```

## Run
```
bash bash/run.sh
```
